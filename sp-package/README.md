# SP Package (Moderate Impact)

Place the official GovRAMP/FedRAMP Moderate Impact SP Package here.

Download from [govramp.org](https://govramp.org). The package contains:

- `1_OCM/` — Official Controls Matrix
- `2_Diagrams/` — Authorization boundary diagram guidance
- `3_Policies/` — 18 policy templates (AC through SR families)
- `4_Procedures/` — 18 procedure templates
- `5_User Guide/`
- `6_Rules of Behavior/`
- `7_Plans Catalog/` — CMP, IRP, ISCM, ISCP, SCRM templates
- `8_Continuous Monitoring Matrix/`
- `9_Laws & Regs/`

The agentic workflow reads the relevant policy template from `3_Policies/` before generating any control narrative.
